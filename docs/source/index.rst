lifx-plugin documentation
========================

A library which translates **Automate Home Appliance's state changes** into *Lifx commands* and
**Lifx bus events** into *messages for Automate Home Appliances*.

An **Appliance** is a non deterministic state machine, abstract representation of one or more physical devices.
